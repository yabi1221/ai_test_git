# user-storage-jpa-master 추가 분석

## 목적

이 문서는 `user-storage-jpa-master`가 실제 운영 저장소인지, 참고/실험용 저장소인지 판단하기 위한 통합 재평가 문서다.

## 관찰 포인트

- Keycloak 사용자 저장소 확장 예제가 풍부하다.
- GPKI/인증 관련 구현이 넓게 섞여 있다.
- 문서와 샘플, 실험 흔적이 `keycloak` 저장소보다 많다.

## keycloak 프로젝트와의 비교

두 저장소 모두 아래 축을 가진다.

- JDBC 사용자 저장소
- GPKI 계열 인증기
- 세션 제한 이벤트

하지만 현재 분석 기준으로는 `keycloak` 쪽이 더 운영형에 가깝고, `user-storage-jpa-master`는 원형/실험 저장소 성격이 더 강하다.

## 실무 해석

이 저장소는 아래 용도로 보는 것이 안전하다.

- Keycloak 사용자 저장소 확장 예제/원형 참고
- 과거 시도나 별도 실험 흔적 확인
- 현재 `keycloak` 저장소와 비교 분석

## 결론

로그인 장애의 1차 조사 대상이라기보다는, 구조 비교용 보조 저장소에 가깝다.
